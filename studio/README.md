# Accordia Studio

Questo laboratorio esiste per farti lavorare con Piny dentro il repo senza toccare subito il sito statico.

## Cosa trovi qui

- una mini app `React + Vite + Tailwind`
- una home editoriale di studio
- una timeline editoriale di studio
- supporto `@pinegrow/piny-vite` gia agganciato al dev server

## Avvio rapido

```bash
npm install
npm run dev
```

Apri poi in VS Code uno dei file dentro `src/pages/` o `src/components/` e usa Piny sulla preview del server locale.

## Route

- `/`
- `/timeline`

## Nota importante

Le modifiche qui dentro non cambiano automaticamente `index.html`, `timeline/index.html` o `css/style.css`.
Questo spazio serve per costruire e ritoccare il design in modo visuale, poi possiamo riportare con precisione i cambiamenti nel sito statico principale.
